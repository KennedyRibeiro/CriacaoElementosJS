Criação de Elementos Dinâmicos com JS

Modelo 1 - Criar elementos formatados

Implemente uma função em JavaScript com o nome criarRetangulos() que toda a vez que o usuário clicar no botão “Criar retângulos” na interface web seja criado um novo retângulo dentro da tag div com o id= “retangulos”. O retângulo deve possuir: 1. 200px de largura; 2. 100px de altura; 3. 10px de margem; 4. Flutuação da esquerda para a direita; 5. Classe com o nome “retangulo”; 6. Cor #736598.

Modelo 2 - Criação de links

Desenvolva uma função em JavaScript chamada de criarLinks() que permita ao usuário adicionar links na tela de acordo com a necessidade. O usuário irá preencher os campos com o nome do link e o endereço (URL) do link. Ao clicar em “Adicionar link”, a função criada deverá adicionar um link como um item na tabela com o id = “links”. O link deverá possuir o nome informado pelo usuário e ao clicar nele, ele deverá redirecionar para a URL informada pelo usuário ao criar o link.

Modelo 3 - Formatar elementos já existentes

Construa uma função em JavaScript com o nome trocarCores() que realiza a troca das cores das formas selecionadas pelos usuários. O usuário irá selecionar uma forma e uma cor e ao clicar em “Trocar cor” a forma escolhida deverá assumir a cor selecionada pelo usuário. A troca não poderá ser feita e o usuário deverá ser alertado se o campo forma e/ou cor não forem preenchidos.
